# Fruit App

A modern e-commerce application built with React for buying fruits and vegetables online.

## Features

- Browse and shop for a variety of fruits and vegetables
- Shopping cart functionality with Redux state management
- User-friendly interface with Material-UI components
- Responsive design using Tailwind CSS
- Product details and checkout process
- Testimonials and featured products section

## Technologies Used

- **Frontend**: React 18.3.1
- **State Management**: Redux Toolkit 2.2.5
- **UI Library**: Material-UI 5.15.20
- **Routing**: React Router DOM 6.23.1
- **Styling**: Tailwind CSS 3.4.4
- **Build Tool**: Create React App (React Scripts 5.0.1)
- **Other**: React Slick for carousels, Emotion for styling

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd fruit-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

   The app will run on `http://localhost:3000`.

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects from Create React App (irreversible)
